# ReactJS - Sample Blog App
> An example blog application that integrates api data with Redux state management using it's lifecycle process.

## Technologies
> Project is created with:  
  * React.js 16
  * ReactDOM 16
  * Lodash 4
  * Redux 4
  * Redux Thunk 2.3

## Setup
```
$ npm install
$ npm run dev

$ yarn install
$ yarn start
```
